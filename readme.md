User can view:

- air pressure = hPa
- air temp = celsius
- cloud fraction = %
- Amount (mm)
- Humidity = "%"
- wind direction = degrees
- wind speed = m/s

- Weather type
- Time

API:
"https://api.met.no/weatherapi/locationforecast/2.0/compact?lat=$(x)&lon=(y)",
"https://nominatim.openstreetmap.org/reverse?format=jsonv2&lat=(x)&lon=(y)"
